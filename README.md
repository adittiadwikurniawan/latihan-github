# Latihan PBO - React

Repository ini digunakan untuk tugas/latihan Pemrograman Berbasis Objek (PBO) menggunakan React.

## Cara Menjalankan Project (Untuk Anggota Tim)

1. Clone repository ini ke laptop masing-masing:
   `git clone https://github.com/adittiadwikurniawan/latihan-github.git`
2. Buka folder project di VS Code.
3. Buka terminal, lalu ketik perintah ini untuk menginstal module bawaan:
   `npm install`
4. Untuk menjalankan website di browser lokal, ketik:
   `npm start`

Mari kerja sama yang baik, teman-teman! 🚀
